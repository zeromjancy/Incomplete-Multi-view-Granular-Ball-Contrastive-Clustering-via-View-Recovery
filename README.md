# Incomplete Multi-view Granular Ball Contrastive Clustering via View Recovery (IMVGCR)

This project implements the **IMVGCR** (Incomplete Multi-view Granular Ball Contrastive Clustering via View Recovery) algorithm for clustering tasks on **incomplete multi-view data**. The core idea is to recover missing views through a View Recovery mechanism and improve clustering performance via Granular Ball contrastive learning.
